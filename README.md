# PostForge - AI Social Post Generator

PostForge is a free, AI-powered tool to generate social media posts for Twitter, LinkedIn, Instagram, and Facebook.

**Live Demo:** [https://evaolennikov-blip.github.io/postforge-mvp/](https://evaolennikov-blip.github.io/postforge-mvp/)

## Features
- **Multi-Platform Support:** Optimized for Twitter (280 chars), LinkedIn (3000 chars), Instagram (2200 chars), and Facebook.
- **Tone Customization:** Choose from Professional, Casual, Humorous, Inspirational, or Persuasive.
- **Context Awareness:** Provide additional context to refine the output.
- **One-Click Copy:** Easily copy generated posts to your clipboard.
- **Free to Use:** Generates 5 posts per day for free.

## How to Use
1. Enter your topic.
2. Select the target platform.
3. Choose a tone.
4. (Optional) Add context.
5. Click "Generate Post".
